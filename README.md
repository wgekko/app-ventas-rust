# app-ventas-rust
app de ventas aplicando rust

para poder ejecutar esta app
instalar las librerias necesarias para esta app
escribir en la terminal esta linea 
pip install -r requirements.txt
se debe abrir una terminal y escribir 
cargo build --release  ## para que corra o se despliegue Rust o
se ejecute el motor de Rust
y luego escribir en la terminal
streamlit run app.py ## para que se ejecute o corra la app
